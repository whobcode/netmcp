netmcp

An authenticated Model Context Protocol server — gives AI agents a toolbox of OSINT, vulnerability-intelligence, ExploitDB search and headless-browser capabilities.

operational GitHub OAuth protected Streamable-HTTP + SSE

MCP endpoint

https://netmcp.hwmnbn.me/mcp

Connecting triggers a one-time GitHub OAuth approval. The legacy SSE transport is at /sse.

Add to an MCP client

{
  "mcpServers": {
    "netmcp": {
      "command": "npx",
      "args": ["mcp-remote", "https://netmcp.hwmnbn.me/mcp"]
    }
  }
}

Works with Claude Desktop / Code, Cursor, and any MCP-compatible client.

Capabilities

🔎 OSINT & reconHost/IP intelligence, DNS history, search & enrichment across external providers.
🛡️ Vulnerability intelCVE lookup, OSV scanning and vulnerability research helpers.
